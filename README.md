
# Kotlin API 테스트 프로젝트

이 프로젝트는 Kotlin과 Spring Boot를 활용한 REST API 개발 및 테스트를 위한 샘플 프로젝트입니다.

## 목적

- 코틀린 언어 기반의 API 서버 개발 연습
- Spring Boot 환경에서 코틀린과 JPA 사용 실습
- WebClient로 kakao API 호출하기(blog search) 
- 인증, 구조 설계, 배포 등의 백엔드 핵심 구성 요소 테스트

## 사용 기술

- Kotlin
- Spring Boot
- Spring Data JPA
- mariadb
- Gradle
